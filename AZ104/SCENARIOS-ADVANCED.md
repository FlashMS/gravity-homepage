AZ-104 Advanced Scenarios

Scenario 16: NSG vs Azure Firewall Conflict
Traffic is blocked even though NSG rules allow it. Diagnose Azure Firewall DNAT/SNAT, rule collections, and forced tunneling.

Scenario 17: Private Endpoint Breaks Public Access
A storage account becomes unreachable after enabling private endpoints. Validate DNS zone overrides, NIC configuration, and firewall rules.

Scenario 18: VM Boot Diagnostics Failure
Boot diagnostics shows no logs. Check managed storage account, permissions, and diagnostic profile configuration.

Scenario 19: App Service Identity Failing
App Service cannot access Key Vault. Validate system-assigned identity, RBAC, access policies, and firewall restrictions.

Scenario 20: Azure SQL Connectivity Issues
SQL database unreachable. Diagnose firewall rules, private endpoints, connection strings, and TLS settings.

Scenario 21: Load Balancer Health Probe Failing
Backend VMs marked unhealthy. Validate probe port, NSG rules, backend NIC configuration, and custom routes.

Scenario 22: Azure Policy Blocking Deployment
A resource deployment fails due to policy. Identify the policy, parameters, exemptions, and remediation tasks.

Scenario 23: VM Extension Errors
Extensions fail to install. Diagnose extension logs, agent health, identity permissions, and network restrictions.

Scenario 24: Storage Account Replication Change Failure
Cannot change replication tier. Validate unsupported features, immutable policies, and soft-delete settings.

Scenario 25: AKS Node Scaling Issues
AKS cluster fails to scale. Check VM quota, subnet IP exhaustion, and node pool configuration.
