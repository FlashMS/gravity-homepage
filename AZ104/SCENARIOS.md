AZ-104 Scenarios

Scenario 1: VM Connectivity Failure
You deploy a VM but cannot RDP or SSH into it. Diagnose NSG, subnet, NIC, and route table issues.

Scenario 2: Storage Account Access Denied
A user cannot access a storage account. Validate RBAC, shared keys, SAS tokens, and firewall rules.

Scenario 3: VNet Peering Not Working
Two VNets are peered but resources cannot communicate. Check DNS, UDRs, and peering configuration.

Scenario 4: Backup Job Failing
Azure Backup fails for a VM. Validate backup vault, policy, extension, and permissions.

Scenario 5: App Failing After Scale Operation
An app breaks after scaling. Validate identity, managed identity permissions, and configuration drift.
